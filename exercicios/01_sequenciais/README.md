# 01 - Sequenciais (Python)

Esta pasta contém exercícios iniciais de programação em Python, focados em estruturas sequenciais (execução linear do código, sem uso de condições ou repetições).

---

##  Objetivo

Praticar os conceitos básicos de programação, como:

- Entrada de dados (`input`)
- Saída de dados (`print`)
- Variáveis
- Operações matemáticas simples
- Formatação de strings

---

##  Conteúdo

Nesta pasta estão 13 exercícios (`ex001.py` até `ex013.py`), desenvolvidos no início do aprendizado de Python durante o curso.

Exemplos de temas abordados:

- Cálculo simples (soma, média, conversões)
- Manipulação básica de valores numéricos
- Uso inicial de `input` e `print`
- Formatação de saída

---

##  Observação

Esses exercícios foram feitos no início do curso e estão sendo mantidos para fins de **evolução e refatoração** posterior, permitindo comparação entre versões iniciais e melhorias futuras no código.

---

##  Conceito

Estruturas sequenciais são programas que executam instruções em ordem, sem desvios condicionais (`if`) ou repetições (`for`, `while`).
