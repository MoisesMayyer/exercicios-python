# 02 - Estruturas Condicionais (Python)

Esta pasta contém exercícios iniciais de programação em Python, focados no uso de estruturas condicionais, permitindo a tomada de decisões no fluxo do programa.

---

##  Objetivo

Praticar conceitos fundamentais de lógica condicional, como:

- Estruturas `if`, `elif` e `else`
- Comparação entre valores
- Operadores relacionais (>, <, ==, !=, >=, <=)
- Tomada de decisão baseada em entrada do usuário
- Controle de fluxo do programa

---

##  Conteúdo

Nesta pasta estão 17 exercícios  (`ex001.py` até `ex017.py`), desenvolvidos durante a fase de aprendizado de estruturas condicionais em Python.

---

##  Exemplos de temas abordados

- Verificação de maioridade
- Cálculo de médias com aprovação/reprovação
- Classificação de valores (positivo, negativo, zero)
- Descontos condicionais em produtos
- Regras simples baseadas em entrada do usuário
- Comparação entre números

---

##  Conceito

Estruturas condicionais permitem que o programa execute diferentes blocos de código dependendo de condições lógicas.

Exemplo:

```python
if condição:
    # executa se verdadeiro
else:
    # executa se falso
